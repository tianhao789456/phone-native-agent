# Publish Checklist

## Before posting

- Confirm `README.md` renders correctly on GitHub.
- Confirm `docs/assets/mobile-agent-home.png` shows the current Chinese app UI.
- Confirm `docs/assets/github-social-preview.png` is set as the repository social preview.
- Confirm latest release notes match the current build.
- Confirm no API keys, runtime tokens, task logs, or private screenshots are committed.

## Suggested order

1. GitHub repository cleanup and release.
2. X / Twitter short post with screenshot.
3. Zhihu short article or answer.
4. V2EX post.
5. Reddit post if account is available.
6. Add to relevant awesome lists later, after at least one stable release.

## Do not overclaim

Use these labels:

- alpha
- experimental prototype
- phone-resident agent
- Android Host App
- AccessibilityService based tools
- SSH/MCP PC bridge

Avoid these labels for now:

- production-ready
- fully autonomous
- secure remote control
- local model included
- no setup required
